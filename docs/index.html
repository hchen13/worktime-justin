输入衰减：连续重复同一个键的反馈逐步变弱，换键后恢复，保留即时感但不奖励乱按。

设计原则

主线是自由探索

孩子不需要理解任务列表、文字说明或复杂模式。默认状态可以随便按键和移动鼠标，反馈轻、即时、好玩。

已定

大反馈只给有意义事件

秘密词、有效探索里程碑、小任务完成、五槽满、连续三次任务完成，才触发明显奖励。乱按也有反馈，但会衰减。

已定

画面必须能回到干净

奖励倾向一次性：烟花、短动画、大贴纸弹出后淡出、临时背景变化。避免永久堆贴纸导致画面越来越乱。

已定

键盘自由探索

普通字母与数字

  • 每按一个普通字母，屏幕随机位置弹出这个字母。
  • 颜色、大小、旋转、位置都可以变化，但必须保证深色背景上高对比可读。
  • 出现方式是“啪”一下弹出，然后约 0.8-1.5 秒逐渐淡出。
  • 字母建议用 SVG / Canvas / HTML text 动态生成，不为每种颜色准备贴图。

功能键与重复按键

  • 空格、回车可以有轻微波纹或弹跳；Command / Option / Control / Shift 反馈很弱或不计奖励。
  • 连续乱按功能键，反馈快速衰减到几乎没有。
  • 长按一个键不持续计数。
  • 连续重复同一个键超过 3 次后暂停计数；换键后再切回来可以重新计数。
  • 正常双写不惩罚，例如 apple 里的 pp 可以成立。

秘密词与五个发现槽

1. 自由输入

系统监听最近的普通英文字母流，不要求输入框，也不要求回车。

2. 命中暗语

例如 xxdogxx 中间出现 dog，或 dogg 在第三个字母 g 时触发 dog。

3. 出现对象

dog 出现小狗、apple 出现苹果，同时播放授权音效或预生成语音。

4. 点亮五槽

当前五格内不重复。同一个词重复命中只给小反馈，不再占新格。

5. 开宝箱

五格满后宝箱打开，触发一次性大奖励，随后进入下一轮。

发现槽来源

五槽不只服务秘密词。它也可以被自由探索里程碑点亮，例如累计 100 次有效按键、200 次有效按键。这样孩子即使没有拼出单词，也会因为持续探索获得小目标。

建议把秘密词命中显示为对应对象图标，把键盘里程碑显示为“键盘星星”这类抽象图标。当前一轮内同类发现不重复。

词池规模

第一阶段按 26 个字母，每个字母约 4 个儿童英语常见词组织，目标约 100 个词。X / Y / Z 这类难字母可减少数量或选择学习场景里常见词。

dog cat apple ball moon star car zoo

问号小任务

入口和退出

  • 默认右侧只保留一个低调的问号，不再放 3-4 个图标按钮。
  • 点问号后播放语音任务，不显示中文任务文字。
  • 15 秒未完成：轻提示一次。
  • 30 秒未完成：目标变明显，比如闪一下或稍微放大。
  • 45-60 秒仍未完成：任务自动收起，不算失败。
  • 如果孩子明显转去玩键盘，例如连续 20 个有效键，任务也自动淡出。

按键

只要求一个键，例如 “Press A.”、“Press 3.”、“Press space.” 不做复杂组合键。

鼠标与触控板反馈

输入 正常反馈 衰减规则 大反馈条件
移动 很淡的光点尾迹，快速移动时稍明显。 连续乱晃约 3 秒后尾迹变弱；停一下再恢复。 经过有效对象可让对象轻微躲开、旋转或发光。
点击 第一下有小星点、短音效或小印章。 连续狂点反馈越来越弱，太快时不给声音。 点中任务目标、宝箱或有效对象时才有明显反应。
拖拽 只有可拖对象进入强反馈，对象弹性跟随。 拖错不惩罚，只轻轻弹回。 拖到正确目标后出现成功动画和任务计数。

奖励系统

五槽满：开宝箱

宝箱奖励以一次性为主，不长期占屏幕空间。适合做烟花、大贴纸弹出后淡出、短动画、临时背景变化、新音效。开完后清空五槽，进入下一轮。

烟花建议用 Canvas / SVG 代码生成，预设“满天星、打铁花、圆形、星形”等类型；颜色从少量高质量色板出发做 HSL / HSV 微调，不做完全 RGB 随机。

任务完成：工作状态灯

小任务奖励不和底部五槽混用。角落放一排很小的工作状态灯，完成一个任务亮一个。连续完成 3 个任务，触发“今日工作完成”奖励。

奖励可以是三个灯一起闪、工作台盖章、小火箭发射、宝箱小开一次。表现要像爸爸工作界面的状态灯，但是儿童版。

桌面应用与安全边界

运行环境

目标机器是 2014 款 MacBook Air。应用应以 macOS 桌面全屏方式运行，并能打包成可安装/启动的应用形式，例如 DMG 或等价分发方式。

Electron 是否可接受需要 TL 做性能验证；也可以评估更轻的原生方案。需求层只要求孩子能进入一个稳定全屏安全空间。

退出与快捷键

  • App 内部拦截 Command+H / Command+W / Command+Q 等常见快捷键。
  • Esc 不直接退出；家长退出可设计为长按 Esc 5 秒后输入口令。
  • Command+Space / Command+Tab 等 macOS 系统级快捷键，普通应用未必能 100% 禁掉。
  • 如需更强控制，需要系统设置、辅助功能权限或家长控制配合，不能在产品需求里虚假承诺。

待确认与后续拆卡

待确认

  • 第一版技术壳：Electron、Swift/Objective-C，或其他轻量桌面方案。
  • 2014 MacBook Air 上的性能预算：动画数量、分辨率、音频加载、全屏稳定性。
  • TTS 模型、授权音效来源、缓存策略。
  • 第一批词池和每个词对应素材。
  • 宝箱、贴纸、状态灯的最终视觉风格。

明显不做

  • 默认界面不做复杂工作台、密集面板、很多按钮或收藏栏。
  • 任务不依赖中文文字说明。
  • 不把失败、惩罚、错误提示作为孩子能感知的主体验。
  • 不为了奖励把贴纸永久堆在主画布上。
  • 不承诺普通应用可以完全屏蔽所有 macOS 系统级快捷键。
