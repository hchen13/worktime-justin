# WTJ-20260703-006 Visual Style Tile

## Goal

Define a compact visual style baseline for WorkTime Justin so the docs and later app work share one direction.

## Output

Save one style tile under:

`/Users/claire/Documents/worktime-justin/docs/assets/style/visual-style-tile.png`

## Required Content

The style tile should show:

- Dark fullscreen canvas background sample.
- High-contrast letter color palette.
- 3-5 example dynamic letters with different sizes and rotations.
- Five discovery slot states: empty, filled with secret-word object, filled by exploration milestone.
- One small `?` task button.
- Three work status lights: off, on, completed streak.
- Treasure chest/reward direction.
- Short visual note areas are allowed outside the app screenshot, but the app UI itself should not rely on text.

## Style Rules

- Premium children's software, not worksheet, not busy dashboard.
- Avoid one-note purple/blue gradients.
- Avoid permanent sticker clutter.
- Keep corners modest, no oversized cards inside cards.
- Use clear contrast on dark background.

## Prompt Starter

```text
Use case: visual style tile for a children's fullscreen desktop app.
Create one polished style tile for "Work Time, Justin!" / "小小工作台".
Show a dark clean canvas sample, high-contrast playful letter colors, dynamic letter examples, discovery slots, a tiny question mark task button, work status lights, and a treasure chest reward style.
The direction should feel like premium calm children's software with a subtle "parents working at a computer" flavor.
Avoid clutter, worksheet style, marketing page layout, large text instructions, copyrighted characters, and low-contrast colors.
```

## Acceptance Notes

This asset is for alignment. It can include labeled sections outside the app frame, but the product UI examples inside the frame should remain minimal.
